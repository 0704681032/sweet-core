Sweeten your JavaScript

Sweet.js brings hygienic macros from languages like Scheme and Rust to JavaScript. Macros allow you to easily extend the syntax of JavaScript and create the language you've always wanted.

Wish function wasn't so long? What if you could define functions like this: 


def sweet(a) {
  console.log("Macros are sweet!");
}

Macros let you do this! 


macro def {
  case $name:ident ($params:ident (,) ...) 
                   {$body:SourceElements} => {
    function $name ($params (,) ...) {
      $body
    }
  }
}

Want a better way to make "classy" objects? 


class Person {
  constructor(name) {
    this.name = name;
  }

  say(msg) {
    console.log(this.name + " says " + msg);
  }
}
var bob = new Person("Bob");
bob.say("Macros are sweet!");

We can do this with macros too! 


macro class {
  case $name:ident { 
    constructor ($cparams:ident (,) ...) 
                {$cbody:SourceElements}
    $($mname:ident ($mparams:ident (,) ...) 
                   {$mbody:SourceElements}) ... 
  } => {

    function $name ($cparams (,) ...) {
      $cbody
    }

    $(
    $name.prototype.$mname = function ($mparams (,) ...) {
      $mbody
    }) ...

  }
}
